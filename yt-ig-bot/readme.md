# Telegram Bot - YouTube & Instagram Downloader

## توضیح پروژه
این ربات تلگرام لینک یوتیوب یا اینستاگرام رو می‌گیره و ویدیو یا صدا (MP3) رو برمی‌گردونه.

## نصب
1. Node.js و npm نصب باشه
2. فولدر پروژه بسازید و فایل‌ها رو کپی کنید
3. نصب پکیج‌ها:
   npm install
4. توکن ربات خودتون رو در bot.js جایگزین کنید
5. اجرای ربات:
   npm start

## نحوه استفاده
- /start → راهنمای ربات
- ارسال لینک یوتیوب → ویدیو برمی‌گرده
- /audio لینک یوتیوب → فقط صدای MP3 برمی‌گرده
- لینک اینستاگرام → نسخه تست، پیام اطلاع
